## Acknowledgements
<p>Built a full-stack web app to allows educators to add courses that are made up of various lessons, while students can enroll in these courses. Additionally, the application will allow students to track their progress throughout the course, whereas instructors can monitor how many students have enrolled in/on a course, and how many have completed each course. The completed application, with all these features, will end up with a home page.</p>

<br />
<p><strong>Used:</strong> NodeJs, ExpressJs, MongoDB, Mongoose, ReactJs, Material-UI.</p>
